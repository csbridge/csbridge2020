Let's Do It Live! : Uploading your Website

Created by Brian Lam

One-time setup

This first part is a little tedious, but don't worry, you only have to do it once.

1.Create a Github account at Github.com

2.Create a new repository

3.Name of the repository must be .github.io (Example provided: my username is blamblam, the repository name is blamblam.github.io) This is important!

4.Click Settings

5.Click automatic page generator

6.Click "Continue to Layouts"

7.Click Publish Page

8.Click index.html

9.Click Edit this file

10.After making some changes (select all and replacing with your index.html file) make sure "Commit directly to master branch" is selected and click "Commit Changes"

After a few minutes, you should see your page live at <your username>.github.io. If it is still not up after 20 minutes, let us know!

If you want to edit the page in the future

Go to http://github.com and click on the repo name <your username>.github.io. After this, from here, it is the same as steps 9 and 10
