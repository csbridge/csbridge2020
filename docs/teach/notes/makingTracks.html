SL Notes: Making Tracks

The problem below is the first lab problem for the day. It is very similar to the lecture examples and meant as a warmup with mouse events. They will need to add mouse listeners in their run method, and then add a bear track at each mouse click location.

Solution

public class MakingTracks extends GraphicsProgram {
    
    public void run() {
        addMouseListeners();
    }
    
    public void mouseClicked(MouseEvent e) {
        drawBearTrack(e.getX(), e.getY());
    }
