Moon Weight Solution

This program gives students practice with variables, user input and printing out to the screen.

public class Moon extends ConsoleProgram {
    
    public void run() {
        double earthWeight = readDouble("Enter a weight on earth: ");
        double moonWeight = earthWeight * 0.165;
        println("Your weight on the moon is " + moonWeight);
    }

}

Random Numbers Solution

This program gives students practice with using control flow in Java, as well as generating random numbers.

public class RandomNumbers extends ConsoleProgram {
    
    // A random number generator
    private RandomGenerator rgen = new RandomGenerator();

    public void run() {
        for (int i = 0; i < 1000; i++) {   
            int randomNumber = rgen.nextInt(0, 100);
            println(randomNumber);   
        }

    }
}
