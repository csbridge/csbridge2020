This is the first lab problem for the morning. It gives students practice writing their first programs with ArrayLists. They need to combine arraylists with user input and printing to print out the numbers the user enters in sorted order.

Solution

public class SortedNumbers extends ConsoleProgram {
    
    private static final int NUMBERS_TO_READ = 10;
  
    public void run() {
        ArrayList<Integer> numbers = new ArrayList<Integer>();
    
        // Read 10 numbers from the user
        for (int i = 0; i < NUMBERS_TO_READ; i++) {
            int number = readInt("> ");
            numbers.add(number);
        }
    
        Collections.sort(numbers);
    
        // Print out the sorted numbers
        for (int i = 0; i < numbers.size(); i++) {
            println(numbers.get(i));
        }
    }
    
}
