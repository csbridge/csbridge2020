Örnek: Kar
Ders Notları: Array List'ler

Göreviniz

loadSecret() isimli metodu kullanabilirsiniz. Bu metod ArrayList<String> biçiminde gizli bir mesaj dönüyor. ArrayList'teki her bir String mesajdaki bir kelimeye karşılık geliyor.

Gizli mesajı görmek için bu array list'teki her bir string'i yazdırın.

Ders Tekrarı

ArrayList üzerinde gezinmek için, for döngüsü kullanın. Örneğin, aşağıdaki kod bir Double ArrayList'indeki tüm değerlerin toplamını hesaplıyor.

ArrayList<Double> values = getValues();
int sum = 0;
for(int i = 0; i < values.size(); i++) {
   sum = sum + values.get(i);
}
println(sum);
