Fibonacci (ve çözümü)

Problemi yazan: Eric Robets

13. yüzyılda, İtalyan matematikçi Leonardo Fibonacci, tavşan popülasyonun geometrik büyümesini açıklama yöntemi olarak, onun ismini taşıyan bir matematiksel seri tasarladı. Serinin ilk iki terimi, Fib(0) ve Fib(1), 0 ve 1'dir, ve sonraki her terim kendisinden önce gelen ikisinin toplamıdır. Böylece, Fibonacci serisindeki ilk birkaç terim şöyledir:

Fib(0) = 0
Fib(1) = 1
Fib(2) = 1 (0 + 1)
Fib(3) = 2 (1 + 1)
Fib(4) = 3 (1 + 2)
Fib(5) = 5 (2 + 3)

Fibonacci serisindeki terimleri gösteren bir program yaz. Fib(0) ile başlasın ve terimler 10.000'den az olduğu sürece devam etsin. Böylelikle, programınız şu örnek çıktıyı üretmeli:

Bu program, terimler maksimum değerden az olduğu sürece çalışmalı. Bunu yapmak için, bir while döngüsü kullanmalısınız. Muhtemelen şöyle bir koşul satırına sahip olur:

while (term < MAX_TERM_VALUE)
Maksimum terim değerinin tanımlanmış bir sabit (constant) kullanılarak belirtildiğini dikkate alın.

Çözüm

/*
 * File: Fibonacci.java
 * --------------------
 * This program lists the terms in the Fibonacci sequence up to
 * a constant MAX_TERM_VALUE, which is the largest Fibonacci term
 * the program will display.
 */

import acm.program.*;

public class Fibonacci extends ConsoleProgram {


   /* Defines the largest term to be displayed */
	private static final int MAX_TERM_VALUE = 10000;

	public void run() {
		println("This program lists the Fibonacci sequence.");
		int t1 = 0;
		int t2 = 1;
		while (t1 <= MAX_TERM_VALUE) {
			println(t1);
			int t3 = t1 + t2;
			t1 = t2;
			t2 = t3;
		}
	}

}
