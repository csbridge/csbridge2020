This is the first section problem for the morning. It gives students practice writing their first graphics programs. There is a little animation mixed in, but that will be covered in more detail in the afternoon.

Teaching tips:

  1. Learning goal: graphics and animation
  2. Milestone: drawing a centered square
  3. You should briefly introduce the pause() function, which delays the program by a specified amount of time.
  4. The hardest part is implementing animation using loops and delays, and connecting this with the graphics programs they have already written.

Solution

public class MysterySquare extends GraphicsProgram {

    private static final int SIZE = 400;
    private static final int DELAY = 1000;

    private RandomGenerator rgen = new RandomGenerator();

    public void run() {
        // Draw a square in the center of the screen.
        GRect rect = new GRect(SIZE, SIZE);
        double x = (getWidth() - rect.getWidth()) / 2;
        double y = (getHeight() - rect.getHeight()) / 2;
        rect.setFilled(true);
        add(rect, x, y);

        // Change its color every second.
        while(true) {
            rect.setColor(rgen.nextColor()); 
            pause(DELAY);
        }
    }
}
